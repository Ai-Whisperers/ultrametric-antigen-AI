## Search Strategy Template

For each topic, use this search pattern:

```
1. PubMed: https://pubmed.ncbi.nlm.nih.gov/?term=[TOPIC]+2024+2025
2. arXiv: https://arxiv.org/search/?searchtype=all&query=[TOPIC]
3. bioRxiv: https://www.biorxiv.org/search/[TOPIC]
4. Google Scholar: https://scholar.google.com/scholar?q=[TOPIC]+2024
5. Semantic Scholar: https://www.semanticscholar.org/search?q=[TOPIC]
```

Replace [TOPIC] with specific search terms.

## Quick Reference

### Claim Language Levels
| Level | When to Use | Example Words |
|:------|:------------|:--------------|
| 1 | Mathematically proven | "is", "proves" |
| 2 | Externally validated | "correlates with" |
| 3 | Internally consistent | "suggests" |
| 4 | Hypothesized | "may", "could" |

### Forbidden Terms (Tier 1/2)
- p-adic → "proprietary encoding"
- ternary → "specialized representation"
- hyperbolic → "geometric"

## Summary of Key Discoveries

### Drug Resistance (Stanford HIVDB)
- **Finding**: Hyperbolic distance correlates with resistance level (r=0.41)
- **MDR Markers**: L63P (79.5%), L90M (58.2%), A71V (54.9%)
- **Clinical**: 2,489 high-risk sequences identified (34.8%)

### CTL Escape (LANL)
- **Finding**: HLA-specific escape velocities quantified
- **Protective alleles**: B57, B27 show low escape velocity
- **Application**: Predict immune pressure patterns

### Antibody Neutralization (CATNAP)
- **Finding**: bnAb breadth correlates with centrality
- **Top bnAbs**: VRC01, PGT128, 10-1074
- **Application**: Antibody design optimization

### Tropism (V3 Coreceptor)
- **Finding**: Position 22 is top tropism determinant (novel)
- **Classic 11/25 rule**: Independently confirmed
- **Accuracy**: 85% prediction with geometric features

### Integration
- **Finding**: 387 vaccine targets ranked by stability
- **Top candidate**: TPQDLNTML (Gag, priority: 0.970)
- **Trade-off**: Drug resistance vs immune visibility
